Padding
Padding is a css property. It works from inside. When we use padding it takes space inside the inline or block itself. We can use padding by writting padding-top,padding-right,padding-bottom and padding-left. And we can use it directly by writting padding:0px 0px 0px 0px.
Margin
Margin is also a css property. It used when we want to take spache outsid from inline and block. Writting system is simillar to padding.
